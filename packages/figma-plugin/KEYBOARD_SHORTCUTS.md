# Critique Lens — Figma Plugin Keyboard Shortcuts

> On Windows/Linux substitute **Ctrl** for **⌘**.

| Shortcut | Action | Notes |
|---|---|---|
| `⌘ ↵` | **Capture selected frame** | Exports the currently selected Figma frame. Blocked inside the notes textarea to avoid newline conflicts. |
| `⌘ ⇧ ↵` | **Run preflight analysis** | Submits the captured frame + selected reviewer. Works from anywhere, including while typing notes. |
| `Escape` | **Close plugin** | Closes the plugin panel. |

## Typical workflow

1. Select a frame in Figma
2. Press **`⌘ ↵`** to capture it
3. Choose a reviewer from the dropdown
4. Optionally select a project and PBI
5. Press **`⌘ ⇧ ↵`** to run the analysis — results open in your browser automatically
